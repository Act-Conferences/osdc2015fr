Présentation de OSDC.fr What is OSDC.fr?

L'objectif de cette Open Source Developers Conference (OSDC) est de réaliser une conférence qui soit un lieu de rencontre et d'échange entre les développeurs de différentes communautés, qui ont souvent tendance à s'ignorer voire se défier. Le but est de dépasser les trolls pour constater ce que chacun peut apprendre des autres. Nous reproduisons en réalité une émulation qui existe déjà par ailleurs et qui a été formalisée par des conférences similaires dans plusieurs endroits du monde : http://www.osdcon.org/ This Open Source Developers Conference (OSDC) aims to be a place for meetings and exchanges between developers from different communities, which too often ignore or distrust each other. The goal is to go beyond trolls and see what we can learn from each other. Similar conferences have been organized in several parts of the world: http://www.osdcon.org/

Pour initier cette conférence en France, nous vous proposons de réunir d'abord les communautés autour de la programmation open source innovante tirée par les langages libres tels que Perl, Python, Ruby... For the first conference of this kind in France, we'd like to begin by bringing together communities that do innovative open source programming using free/libre languages such as Perl, Python, Ruby...

Le but est clairement d'apprendre, de voir ce que les autres font afin d'en profiter, car il y a souvent à apprendre des autres en terme de concepts ou de techniques non intrinsèquement liés à un langage ou à un autre (on peut citer de nombreux exemples). The goal is to learn by seeing what others are doing. Many concepts and techniques are not tied to a specific language.

La porte est ouverte à bien d'autres langages et technos, en particulier les plus novateurs ou spécialisés comme Erlang, Haskell, OCaml, Smalltalk, Scala, Groovy, etc. We of course welcome many other languages and technologies, particularly the most innovative or specialized, such as Erlang, Haskell, OCaml, Smalltalk, Scala, Groovy, etc.

Cette conférence vise donc principalement un public de développeurs et utilisateurs avancés de logiciels libres. Le but est de pouvoir parler technique sans pincettes ou frilosité. So this conference mainly targets developers and advanced users of free and open source software. The idea is to be able to talk tech without fear of not being understood.

Un point important : OSDC.fr, initiée par les 3 associations françaises de Perl, Python et Ruby, rejointes en 2011 par le Smalltalk User Group, n'est pas destinée à fusionner les conférences des communautés de langages. Il s'agit de lancer une nouvelle série de conférences ouvertes à l'ensemble des développeurs open source pour mieux échanger. An important note: OSDC.fr, which has initially been founded around the 3 French user groups of Perl, Python, and Ruby (Smalltalk User Group joined in 2011), does not aim to merge the conferences of each language's community. The idea is to launch a new series of conferences, open to all open source developers as a place to exchange information.

L'intérêt de OSDC.fr est de réaliser ce genre de présentations inter-langages dans un cadre neutre, hors du contexte spécifique d'une communauté particulière. The advantage of OSDC.fr is that it is a place to present these kinds of inter-language talks in a neutral setting, outside of the specific context of a particular language community.
